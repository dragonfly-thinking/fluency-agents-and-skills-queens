# McGill citation checklist

What the citation-checker subagent checks on every invocation, against the *Canadian Guide to Uniform Legal Citation* (the "McGill Guide").

## Case citations
- [ ] Style of cause **italicised**; parties joined by *v* (no period, Canadian style)
- [ ] **Neutral citation** included where one exists (e.g. `2019 SCC 65`, `2023 FCA 12`)
- [ ] **Parallel citation** to an official reporter where appropriate (e.g. `[2019] 4 SCR 653`)
- [ ] Correct order and punctuation: *Style of Cause*, neutral cite, parallel cite
- [ ] Year correct and in the right position

## Pinpoints
- [ ] Modern judgments pinpointed to **paragraph** ("at para 85", "paras 85–86") — not page
- [ ] Pinpoint format consistent throughout

## Statutes
- [ ] Title **italicised**
- [ ] Statute volume, year, chapter present: `SC 2001, c 27` (annual) or `RSC 1985, c …` (consolidated)
- [ ] Pinpoint to **section** (`s 3`, `ss 3–4`), not paragraph or page

## Short forms & consistency
- [ ] First reference is full; later references use a consistent short form
- [ ] `ibid` only for the immediately preceding citation
- [ ] `supra note N` points to the correct earlier note
- [ ] The same source cited the same way throughout

## Honesty rules
- [ ] For a well-known authority, the corrected citation may be supplied from knowledge — but marked **"(verify against reporter / A2AJ)"**
- [ ] For an unknown source, show the McGill **template** and flag the gap — never invent a volume, reporter, or paragraph number
- [ ] Comparative US (Bluebook) / UK (OSCOLA) citations are noted as their own systems, not forced into McGill

## Out of scope (hand off)
- [ ] Prose, grammar, tone → `proofread` (writing-editor)
- [ ] Whether the source supports the claim, or a date/holding is right → `critical-review` (fact-checker; checks against A2AJ when connected)
