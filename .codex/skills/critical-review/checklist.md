# Critical Review checklist

What the two subagents check.

## Critical Friend (argument lens)

- [ ] What is the writer claiming? (One sentence)
- [ ] What evidence are they using to support it?
- [ ] What are the hidden assumptions?
- [ ] Where is the reasoning weakest?
- [ ] What is the strongest counter-position a thoughtful opponent would defend?
- [ ] Whose perspective is missing from the argument?
- [ ] What would have to be true for the writer's view to be wrong?

## Fact-checker (evidence lens)

- [ ] What factual or statistical claims appear in the draft?
- [ ] For each numerical claim, what is the actual value (per the primary source)?
- [ ] What is the source publisher, dataset/release, date, and URL?
- [ ] Are time-bound claims using current data, or stale data?
- [ ] Are comparative claims using the right comparison group?
- [ ] Are there claims that can't be verified — why not?

## Boundaries (what neither subagent does)

- Grammar and prose-level editing — handled by the `proofread` skill
- Rewriting the draft — that's the user's job
- Recommending what to do next — they identify; the user decides
